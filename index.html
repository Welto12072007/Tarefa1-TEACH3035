Tags Estruturais

As tags estruturais do HTML são elementos que delimitam diferentes partes do conteúdo em um documento web.

Tags Principais:

  • <header> - Define o cabeçalho
  • <nav> - Contém links de navegação
  • <main> - Conteúdo principal
  • <section> - Agrupa conteúdo relacionado
  • <article> - Conteúdo independente

Exemplo de Tabela

Tipos de Tags HTML
